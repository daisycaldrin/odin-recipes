# Odin Recipes

This project is part of [The Odin Project ](https://www.theodinproject.com/) Foundation course. It's a simple recipe website built with plain HTML to practice structuring a multi-page site and pushing a project to Github using Git version  control.

## Skills demonstrated
- Writing semantic HTML
- Structuring a basic multi-page website (homepage + individual recipe pages)
- Using Git for version control (commits, branches)
- Hosting a project on GitHub
- Deploying a static site with GitHub Pages
